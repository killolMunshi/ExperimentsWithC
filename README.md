# ExperimentsWithC

A C programming project for experimentation and learning.

## Building and Running

### Using GCC (MinGW on Windows)
```bash
gcc main.c -o main.exe
./main.exe
```

### Using Clang
```bash
clang main.c -o main.exe
./main.exe
```

## Project Structure

- `main.c` - Main entry point of the program
